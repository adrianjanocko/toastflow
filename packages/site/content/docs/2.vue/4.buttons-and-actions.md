Add inline action buttons through the `buttons` config.

## Basic Button Group

```ts [action-button.ts]
toast.info({
  title: "File archived",
  description: "Use an action when the user can recover immediately.",
  buttons: {
    alignment: "bottom-right",
    buttons: [
      {
        id: "undo",
        label: "Undo",
        dismissAfterClick: true,
        onClick() {
          toast.success({ title: "Restored" });
        },
      },
    ],
  },
});
```

## Button Alignment

| Value          | Position                       |
| -------------- | ------------------------------ |
| `top-left`     | Above content, left            |
| `top-right`    | Above content, right           |
| `center-left`  | Beside content, left           |
| `center-right` | Beside content, right          |
| `bottom-left`  | Below content, left            |
| `bottom-right` | Below content, right (default) |

## Layout Direction

::tabs
::tabs-item{label="row (default)"}
Action group sits side-by-side with the toast content.

```ts [layout-row.ts]
toast.info({
  title: "Release 1.2.0",
  buttons: {
    layout: "row",
    buttons: [{ label: "Open changelog" }],
  },
});
```

::

::tabs-item{label="column"}
Action group takes a dedicated row above or below content.

```ts [layout-column.ts]
toast.info({
  title: "Release 1.2.0",
  buttons: {
    layout: "column",
    buttons: [{ label: "Open changelog" }, { label: "Remind me later" }],
  },
});
```

## Each Button

Each button supports:

::field-group

::field{name="label" type="string"}
Plain text label.
::

::field{name="html" type="string"}
HTML string (requires `supportHtml` on the toast).
::

::field{name="id" type="string"}
Unique identifier.
::

::field{name="ariaLabel" type="string"}
Accessible label.
::

::field{name="className" type="string"}
Custom CSS class.
::

::field{name="dismissOnClick" type="boolean"}
Dismiss the toast before `onClick` runs.
::

::field{name="dismissAfterClick" type="boolean"}
Dismiss the toast after `onClick` runs.
::

::field{name="onClick" type="(ctx, event) => void"}
Click handler.
::

::

::caution
`html` buttons are rendered without sanitization — pass trusted markup only, or configure the [`sanitizer` hook](/docs/global/styling#sanitizer-hook). Set `ariaLabel` on `html` buttons so screen readers can announce them.
::

## Close Interactions

::field-group

::field{name="closeButton" type="boolean"}
Show or hide the floating close icon.
::

::field{name="closeOnClick" type="boolean"}
Dismiss on toast body click.
::

::field{name="onClick" type="(ctx, event) => void"}
Custom body click callback.
::

::field{name="onClose" type="(ctx) => void"}
Callback fired before the leaving phase.
::

::

```ts [close-interactions.ts]
toast.warning({
  title: "Action required",
  closeOnClick: true,
  onClick(ctx) {
    console.log("Clicked", ctx.id);
  },
  onClose(ctx) {
    console.log("Closing", ctx.id);
  },
});
```

## Created-At Badge

Show a timestamp on the toast:

```ts [created-at.ts]
toast.default({
  title: "Background sync",
  showCreatedAt: true,
  createdAtFormatter(ts) {
    return new Date(ts).toLocaleTimeString();
  },
});
```

If the formatter throws or is missing, Toastflow falls back to its default time formatter.
